vue-mapbox-map


Mapbox GL JS wrapped in a Vue module for dynamic maps.


Add dynamic mapbox instances to your Vue components: 


    <vue-mapbox-map id='map'
                    :access-token='accessToken'
                    :interactive='false'
                    :geocoder='false'
                    :lng='lng'
                    :lat='lat'
                    :zoom='zoom'
                    :pitch='pitch'
                    :bearing='bearing'
    ></vue-mapbox-map>

Scroll to see real-time map updates!
